# Textile Inventory System - Frontend

React + Vite frontend for Textile Inventory & Billing System.

## Quick Start

```bash
npm install
npm run dev
```

Access at: http://localhost:3000

## Login

- Username: admin
- Password: admin123

## Features

- Supervisor Dashboard: Inward lot entry
- Floor Staff Dashboard: Program entry with camera
- Admin Dashboard: Billing & reports
- Mobile camera integration
- Responsive design

Backend must be running on http://localhost:8000
